## About This Library / 關於本圖書館

This library is powered by the **Audio Researcher** automated workflow. Each report goes through a rigorous 7-phase process:

本圖書館由 **Audio Researcher** 自動化工作流驅動。每份報告經過嚴格的 7 階段流程：

1. **Phase 0**: Initialization / 初始化
2. **Phase 1**: Multilingual search (EN/JP/ZH/DE) / 多語系搜尋
3. **Phase 1.5**: URL audit / URL 審計
4. **Phase 2**: Outline planning / 大綱規劃
5. **Phase 2.5**: Claims audit / 引用審計
6. **Phase 3**: Report generation (bilingual EN+ZH) / 報告生成（雙語）
7. **Phase 3.5**: Final audit & quality grading / 最終審計與品質分級

### Quality Grades / 品質評級

| Grade | Meaning / 意義 |
|-------|----------------|
| **A** | >40% L1 (official) sources, no number conflicts, >70% claims verified |
| **B** | >20% L1 sources, ≤2 conflicts resolved, >70% claims verified |
| **C** | Below B criteria — flagged for human review |

### Evidence Levels / 證據等級

| Level | Type / 類型 |
|-------|------------|
| **L1** | Official documentation, patents, peer-reviewed papers / 官方文件、專利、同儕審查論文 |
| **L2** | Professional media reviews with measurements / 含量測的專業媒體評測 |
| **L3** | Professional community consensus / 專業社群共識 |
| **L4** | General content (reference only) / 一般內容（僅供參考） |
